# Changelog — v0.3.0

> **Status:** unreleased changelog draft

## Summary

v0.3.0 captures the Rudolph Beacon benchmark suite, CodeRabbit decommissioning, controlled real-mode probe, portfolio gap discovery, and the Windows tool-gateway repair.

## Rudolph Beacon (#279)

- `packages/benchmark-rudolph/` package added to main (PR #295).
- Controlled real-mode probe (`runControlledRealModeProbe()`) with 5 safety gates.
- Red-negative test suite (36 tests) validating that safety violations are correctly blocked.
- Coverage policy (`COVERAGE_POLICY.md`) enforced.
- Benchmark specification (`BENCHMARK_SPEC.md`) documents the evaluation contract.
- Real mode readiness assessment (`REAL_MODE_READINESS.md`).

## CodeRabbit Decommissioned

- Internal CodeRabbit configuration removed (commit `5494851`).
- External CodeRabbit app removal pending owner action.

## Portfolio Gap Discovery (PR #309)

- Comprehensive audit of 14 open + 91 closed issues.
- 24 capability areas assessed: 4 IMPLEMENTED, 11 PARTIAL, 8 ISSUE_EXISTS, 1 STALE.
- 4 new issues created: #305 (Portfolio Auto-Update), #306 (Backlog Hygiene), #307 (Docs Sync), #308 (Full Real Mode Pilot).
- 14 gaps mapped to existing open issues.

## Windows Tool-Gateway Repair (#299)

- Windows module resolution failure in `packages/tool-gateway` fixed (PR #303).
- Root cause: Node.js `child_process` module path resolution on Windows (`\` vs `/`).
- All 1571 tests pass consistently on Windows.

## Test Reality

- **Total: 1571/1571 tests** (1375 root + 196 apps/web).
- apps/web JSX/TSX issues resolved (all 196 pass).
- 72 test files across all packages.

## Documentation Sync (#307)

- Status docs (`current-capabilities.md`, `known-limitations.md`) updated for post-closeout reality.
- Evidence index (`docs/status/evidence-index.md`) created.
- Changelog v0.2.0 and v0.3.0 (this file) created.
- Limited api-overview sync with note about #251.
